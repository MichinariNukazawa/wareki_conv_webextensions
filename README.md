Daisy WarekiConv - converted from Japanese era WebExtensions
----
In page convert with Japanese calendar to Western calendar.  
ページ内の和暦(昭和・平成など)を西暦に自動変換します。  
変換した西暦は和暦の後ろに挿入して表示されます。
